Proyectos

VialdKids

Plataforma Web interactiva orientada a la educación vial infantil (3 a 8 años), desarrollada como proyecto grupal con enfoque social. La iniciativa combinó una experiencia digital con talleres presenciales en jardines de infantes, orientados a la concientización temprana sobre seguridad vial.

El proyecto fue presentado en ferias de ciencias y en el concurso internacional Solve for Tomorrow de Samsung, alcanzando la tercera etapa. Además del sitio web, se diseñaron actividades lúdicas y un prototipo conceptual de una mini ciudad para que los niños aprendieran mediante el juego.

Mi rol incluyó aportes en Front-End y Back-End, diseño de la experiencia de usuario, planificación del proyecto y desarrollo de contenidos educativos, tanto para la web como para los talleres presenciales.

Turnera Profesional

Aplicación web desarrollada para la gestión de turnos y fichas de pacientes de un profesional de la salud. El sistema permite administrar citas, almacenar historiales y acceder a la información de forma clara y eficiente.

Desarrollé la solución utilizando Django para el backend y HTML/CSS para el frontend, implementando lógica de negocio, persistencia de datos y una interfaz orientada a la usabilidad.
